# frontend
vue cli 3のもりもり設定でプロジェクト作成したのを雛形に作成。
- typescript
- pwa surpprt
- router
- vuex
- unittest

## Project setup
1. install dependencies
    ```
    npm install
    ```
1. modify .env (reffer to `.env.sample`)
1. run api server (refer to `/back/readme.md`)
1. run webpack-dev-server
    ```
    npm run serve
    ```
